# Shark Music Bot

Discord music bot built around Discord.py, YT-DLP, and FFMPEG.

## 🎚️ Features
- Youtube Links
- Youtube search
- Soundcloud links
- Dynamic embeded messages

## ⌨️ Commands
- `/play`: play a YouTube or Soundcloud link, or search for a youtube video by title.
- `/skip`: Skips currently playing song and play the next in queue OR stops currently playing song if nothing in queue.
- `/leave`: Disconnects the bot from voice channel.
- `/clean`: Purges the `limit` amount of bot-related messages in a text channel (Default limit value is 100).
